# Amazon Customer Support Analytics Dashboard | Power BI

## Overview

The Amazon Customer Support Analytics Dashboard is an end-to-end Business Intelligence project designed to analyze customer support operations, monitor agent performance, and provide actionable business insights through interactive data visualization.

This project simulates a real-world customer support environment and demonstrates how data can be leveraged to improve operational efficiency, workforce productivity, service quality, and customer experience. The dashboard enables managers and stakeholders to track key performance metrics, identify performance trends, and make data-driven decisions.

---

## Business Problem

Customer support operations generate large volumes of data daily, making it difficult for managers to monitor performance, identify inefficiencies, and evaluate agent productivity.

This project addresses these challenges by providing a centralized analytics solution that:

* Tracks operational performance in real time.
* Measures agent productivity and efficiency.
* Monitors transfer and park rates.
* Evaluates support quality and chat outcomes.
* Identifies coaching and improvement opportunities.
* Supports data-driven decision-making.
